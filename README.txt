Instrucțiuni rapide
1) Pune toate fișierele într-un folder pe hosting (sau GitHub Pages).
2) Asigură-te că supermarket.jpg este în același folder cu index.html.
3) Deschide index.html.

SEO:
- Fiecare articol are pagină separată + meta description + JSON-LD.
- sitemap.xml este inclus (înlocuiește loc-urile cu domeniul tău real, dacă vrei un sitemap complet).

Formular:
- Momentan afișează un mesaj (nu trimite). Pentru trimitere reală folosește un serviciu tip Formspree sau un backend.
